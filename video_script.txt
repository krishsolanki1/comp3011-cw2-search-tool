[3:00-4:00] CODE WALKTHROUGH
----------------------------------------------------------

Open src/crawler.py.

Say:
"The crawler uses requests and BeautifulSoup. It follows the li.next
pagination link, converts relative URLs to absolute with urljoin, and checks
the netloc matches the base domain to prevent following external links. It
sleeps for six seconds between fetches. The sleep function is injected as a
parameter so tests can replace it with a no-op -- no test ever actually
waits."

Open src/indexer.py.

Say:
"The indexer strips script, style, head, and noscript tags from the HTML then
tokenises the remaining text using the regex [a-z0-9]+ -- which lowercases
everything and drops punctuation in one step. Each token's position in the
document is recorded alongside its frequency, so the index stores both."

Open src/search.py.

Say:
"The search engine normalises queries with the same tokeniser so search is
always case-insensitive. For multi-word queries it intersects the URL sets
for all terms -- AND logic -- then scores each result using TF-IDF. IDF
weights rare terms more heavily, so a page with one occurrence of
indifference can outrank a page with three occurrences of the. The proximity
bonus uses the stored position data to reward documents where query terms
appear close together."

"I used two AI tools in this project with different roles. I started by using

flagged it, and it was fixed. Throughout the project I was noting things I
wanted changed, and at the end we did a final pass to address everything I'd
spotted.

The planning, the reviews, the decisions about what to change, and the final
